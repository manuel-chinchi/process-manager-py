# Process Manager Py
A simple Process manager program made in Python + TKinter

## Libraries

* TKinter v8.6
* psutil v6.1.1
* pyperclip v1.9.0
* diskcache v5.6.3

## Supported platforms

* Windows 10 *(v22H2 c19045.5131)*
